# LinkUsApp
> 스터디 주제가 맞는 사람들끼리 스터디를 할 수 있게 해주는 앱.
- 개발 기간 : 2020.12.24~2021.04.11
- 사용 언어 : Java, NodeJS
- 데이터 베이스 : MySQL
- 소셜 로그인, fcm, 소켓 통신
- `Java`, `MySQL`, `MVVM`, `retrofit2`, `aws RDS`, `aws ec2`, `firebase(fcm)`, `SharedPreferences`
